# 배포 전 디자인 QA 체크리스트 (모바일 우선)

기준일: 2026-03-07
대상: Web + 앱(WebView/PWA) 동시 배포

## 0. 사용 방법
- 각 항목을 `PASS / FAIL / N/A`로 기록
- `P0`에서 1개라도 FAIL이면 배포 중지
- `P1`은 배포 직전 모두 PASS 권장
- `P2`는 배포 후 1주 내 보완

---

## 1. 우선순위 기준

### P0 (배포 차단)
- [ ] 홈/핵심 탭에서 레이아웃 깨짐 없음 (모바일 360px 기준)
- [ ] 주요 버튼 터치 영역 44px 이상
- [ ] 고정 헤더/하단 영역이 본문과 겹치지 않음
- [ ] 로딩/에러 상태에서 사용자 액션(재시도/뒤로) 가능
- [ ] 다크/라이트 모드에서 텍스트 대비 문제 없음

### P1 (배포 강권)
- [ ] 카드/버튼/입력창 radius, shadow, spacing 일관성 유지
- [ ] 스켈레톤/빈 상태/에러 상태 톤 통일
- [ ] 폼 입력 시 키보드 오버랩 없음 (iOS Safari/Android Chrome)
- [ ] 긴 텍스트(닉네임/학교명/주소) 줄바꿈/말줄임 처리 정상
- [ ] 지도/차트/모달 컴포넌트가 작은 화면에서도 사용 가능

### P2 (배포 후 개선)
- [ ] 마이크로 인터랙션(hover/focus/press) 일관화
- [ ] 시각적 밀도(여백/타이포 계층) 미세 조정
- [ ] 운영 화면(admin) 정보 우선순위 정리

---

## 2. 공통 디자인 QA

### 레이아웃/타이포
- [ ] 헤더 높이, 본문 패딩, 카드 간격 규격 통일
- [ ] 제목/본문/보조텍스트 계층이 명확
- [ ] 숫자/통계는 tabular 숫자 사용(표/대시보드)

### 색상/가독성
- [ ] 버튼 primary/secondary/disabled 상태 구분 명확
- [ ] 경고/성공/오류 색상이 기능적으로 일관
- [ ] 배경과 카드 대비가 충분해 정보 구분이 쉬움

### 인터랙션
- [ ] 버튼 연타/중복요청 방지(loading 상태)
- [ ] 네트워크 지연 시 사용자 피드백 제공
- [ ] 모달 ESC/외부 클릭/닫기 버튼 모두 동작

### 반응형
- [ ] 360x800, 390x844, 768x1024, 1440x900에서 깨짐 없음
- [ ] 가로 스크롤이 필요한 경우 의도된 컴포넌트(테이블 등)만 허용
- [ ] Safe area(`env(safe-area-inset-*)`) 적용 확인

---

## 3. 페이지별 체크 (핵심)

### 홈 `/`
- [ ] 상단 네비/탭 전환 시 점프 현상 없음
- [ ] 첫 진입 로딩 후 레이아웃 흔들림 없음

### 커뮤니티 `/community`, `/community/[id]`, `/community/write`
- [ ] 목록 카드 높이/간격 일관성
- [ ] 이미지 업로드/미리보기 비율 정상
- [ ] 댓글/좋아요/신고 버튼 터치 영역 충분

### 음식 `/food`, `/food/result/[id]`, `/food/favorites`, `/food/insights`
- [ ] 검색 입력/자동완성 충돌 없음
- [ ] 결과 카드 긴 상품명 잘림 처리 정상
- [ ] 분석 중/실패 상태 메시지 가독성 확보

### 아파요(병원/약국) 탭
- [ ] 지도 영역이 헤더/하단 버튼과 겹치지 않음
- [ ] 줌/이동 시 과도한 깜빡임/오버레이 없음
- [ ] 마커 선택 시 정보 카드 위치 안정

### 식단 `/diet`
- [ ] 등록/수정/삭제 플로우 버튼 상태 명확
- [ ] 입력 폼 키보드 오버랩 없음

### 학교 `/school`, `/school/[id]`
- [ ] 학교 검색 결과 리스트 줄바꿈 정상
- [ ] 급식 카드에서 긴 텍스트 overflow 처리

### 마이페이지 `/mypage`, `/bookmarks`, `/family`, `/support`
- [ ] 프로필/설정 입력폼 정렬 일관
- [ ] 빈 데이터 상태(북마크 없음 등) 디자인 일관

### 관리자 `/admin`
- [ ] 탭 전환 시 레이아웃 흔들림 없음
- [ ] 데이터 상태 대시보드 표 가독성(모바일/데스크탑)
- [ ] 운영 기능 버튼(초기화/새로고침) 오클릭 방지

## 4. 앱(WebView) 전용 체크

- [ ] iOS Safari + Android Chrome에서 동일 동작
- [ ] 스크롤 bounce/주소창 축소 시 고정영역 깨짐 없음
- [ ] 파일 업로드/카메라 호출 후 복귀 시 화면 상태 유지
- [ ] 외부 링크/딥링크 이동 후 복귀 UX 확인
- [ ] 글자 확대(접근성 폰트 크기)에서도 주요 CTA 노출 유지
